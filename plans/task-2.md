# Task 2: Documentation Agent - Plan

## Tool Design
- **read_file**: читает файл по относительному пути от корня проекта. Безопасность: проверка на directory traversal через `os.path.abspath` и сравнение с корнем.
- **list_files**: возвращает список файлов и папок в указанной директории. Также безопасен.

## Agentic Loop
1. Отправляем пользовательский вопрос + системный промпт (с инструкцией использовать инструменты) в LLM.
2. Если ответ содержит `tool_calls`:
   - Для каждого вызова выполняем соответствующую функцию.
   - Добавляем результат как сообщение с ролью `tool`.
   - Повторяем цикл (до 10 итераций).
3. Если ответ не содержит `tool_calls` — это финальный ответ. Извлекаем `source` (по наличию пути wiki/... в тексте) и выводим JSON.

## Системный промпт
Будет содержать инструкцию использовать `list_files` для обзора wiki, затем `read_file` для чтения нужных файлов. Также требование указывать источник в ответе.
